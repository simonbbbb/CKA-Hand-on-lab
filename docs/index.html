Quick Start

Get up and running in under 5 minutes

1

Clone the Repository

bash
$
git clone https://github.com/simonbbbb/CKA-Hand-on-lab.git
$
cd CKA-Hand-on-lab
2

Start the TUI

bash
$
cargo run --release
3

Or Set Up Manually

bash
$
cd setup
$
./lab_launcher.sh

Prerequisites

  • • Rust toolchain (for TUI) - or use pre-built binaries
  • • kubectl configured and pointing to your cluster
  • • A Kubernetes cluster (minikube, kind, k3s, or cloud)
  • • Basic familiarity with Kubernetes concepts
